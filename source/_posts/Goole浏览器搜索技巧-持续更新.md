# GitHub

* <content> in:file 搜索文件中有content的代码
* <content> in:path 搜索路径中有content的代码
* <content> language:java  搜索用java写的包含content的代码
*  filename:.vimrc commands 搜索 文件名匹配*.vimrc* 并且包含commands的代码
